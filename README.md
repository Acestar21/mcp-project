# Developer Setup Guide

## Prerequisites
1. **Python 3.10+** (Ensure it is in your PATH)
2. **Node.js & npm**
3. **Rust** (Cargo)
4. **Ollama** (Running in background)

## Installation
1. Install Python dependencies:
   ```bash
   pip install mcp[cli] ollama playwright
   playwright install

2. Install Frontend dependencies:
    ```bash
    cd mcp-app
    npm install
